Effective inventory management begins with understanding your starting capacity and the first few options available. <Section icon={Map} title="Co-op Inventory Management and Base Storage">
Subnautica 2's co-op feature introduces a new dimension to inventory management. While personal inventories remain separate, shared storage lockers within bases and shared blueprints make collective resource management more efficient.

<Callout type="info">
Shared storage lockers at launch mean that any item placed in a built locker by one player is accessible to all players in that co-op world. This streamlines resource sharing but requires good communication.
</Callout>

Coordinating storage in a multiplayer game is vital to avoid confusion and ensure everyone has access to necessary materials.

**Co-op Inventory Strategies:**

| Strategy                 | Description                                                 | Benefits                                          | Challenges                                          |
| :----------------------- | :---------------------------------------------------------- | :------------------------------------------------ | :-------------------------------------------------- |
| **Centralized Hub**      | One main base with clearly labeled shared storage for all resources. | Easy access for everyone, reduces duplication.    | Can become cluttered, requires strong organization. |
| **Specialized Storage**  | Designate specific lockers for different resource types (e.g., metals, organics, rare minerals). | Highly organized, quick retrieval of specific items. | Requires initial setup and team adherence.          |
| **Personal "Stash" Areas**| Players can build small, personal lockers near their workstations for private items. | Allows for individual preference, keeps personal gear separate. | Can lead to resource hoarding if not balanced with shared storage. |
| **Dive Elevator Usage**  | Use the Dive Elevator to send gathered resources directly to a central base storage. | Reduces return trips, safer transport of valuable items. | Requires a constructed base with an elevator module. |
