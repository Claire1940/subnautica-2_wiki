### Step-by-Step Installation

<Steps>
  <Step num="1" title="Locate Game Directory">
    Navigate to your Subnautica 2 installation folder. For Steam users, right-click the game in your library, go to "Manage," then "Browse local files."
  </Step>
  <Step num="2" title="Download UE4SS">
    Download the UE4SS for Subnautica 2 zip file from a reputable modding website. This is the foundational mod that allows other mods to function.
  </Step>
  <Step num="3" title="Extract UE4SS Files">
    Go to `\Subnautica2\Subnautica2\Binaries\Win64` within your game directory. Place the downloaded UE4SS zip file here and extract its contents directly into this folder.
  </Step>
  <Step num="4" title="Download Console Commands Mod">
    Download the Console Commands mod zip file. This mod contains the actual console functionality.
  </Step>
  <Step num="5" title="Install Console Commands Mod">
    Open the Console Commands mod zip file and locate the "DebugUIToggle" folder. Place this entire folder into `\Subnautica2\Subnautica2\Binaries\Win64\Mods` within your game directory. If the 'Mods' folder doesn't exist, create it.
  </Step>
  <Step num="6" title="Launch Game & Open Console">
    Start Subnautica 2. Once in-game, press the `F2` key to open the command console. You should now see a variety of options and be able to input commands.
  </Step>
</Steps>

### Item Spawning and Resource Management

One of the most popular uses for console commands is to spawn any item directly into your inventory. This can save immense amounts of time and effort in gathering resources.

| Command Syntax          | Description                                                     |
| :---------------------- | :-------------------------------------------------------------- |
| `item [name] [amount]`  | Adds a specified amount of an item to your inventory.           |
| `spawn [name] [amount]` | Spawns a specified amount of an object in front of the player.  |

**Example:** To add 5 units of Lithium to your inventory, type `item lithium 5`.

### Teleportation Commands

| Command             | Effect                                                              | Example Usage                         |
| :------------------ | :------------------------------------------------------------------ | :------------------------------------ |
| `biome [NAME]`      | Teleports the player to a specific biome.                           | `biome shallowtwistybridges`          |
| `warp [x] [y] [z]`  | Warps the player to specified coordinates.                          | `warp 100 50 200`                     |
| `goto [NAME]`       | Teleports to named locations (e.g., `goto rocket`).                 | `goto rocket`                         |
| `spawnnearby`       | Teleports player a short distance in a random direction (unstuck).  | `spawnnearby`                         |

<Callout type="warning" title="Coordinate System">
When using `warp` commands, remember that `[x]` is East/West, `[y]` is Altitude, and `[z]` is North/South. Incorrect coordinates can place you in dangerous or unintended locations.
</Callout>
