<Section icon={Swords} title="協力プレイのメカニクスと共有リソース">

**Subnautica 2オンライン**では、協力とは単に一緒に泳ぐこと以上のものです。それは共有された生態系を管理することです。ゲームは、複数のプレイヤーに対応するためにリソースが豊富になるように設計されており、材料の不足が進行を妨げないようにしています。ただし、特定のハイテクシステムはロビー全体で共有されるため、パイオニア間のコミュニケーションが必要です。

重要なケースの1つが**スキャナーステーション**です。鉛や銀などの特定のリソースを見つけるのに非常に効果的ですが、ステーションの出力は普遍的です。あるプレイヤーがスキャナーをクォーツを探すように設定すると、セッション内のすべてのプレイヤーがHUDにそのクォーツノードを見ることになります。ステーションごとに一度に追跡できるリソースは1つだけなので、これは個々のツールではなく共有ツールとなります。

<Section icon={Sparkles} title="進行：遺伝的適応とツール">

Subnautica 2の進行は、単純な酸素タンクのアップグレードを超えて進化しました。サバイバルの核は、今や**遺伝的適応**にあります。**適応パビリオン**と深い入り江で見つかる様々な「エンジェルツリー」と交流することで、プレイヤーはDNAを改変し、過酷なエイリアンの環境に耐えることができます。

<Steps>
  <Step num="1" title="マルチツールとスキャナーの入手">
    繊維質のパルプとアシッドポーチを集めて基本的なバッテリーを作成します。スキャナーは、設計図のロックを解除し、地元の動物相を研究するための最も重要なツールです。
  </Step>
  <Step num="2" title="ウェルカムセンターの場所を特定">
    アルテラウェルカムセンターを見つけて、**生息地ビルダー**と**バイオラボ**のロックを解除します。この建物の電源を入れることは、初期のテクノロジーにとって不可欠です。
  </Step>
  <Step num="3" title="最初の適応の獲得">
    初期の難破船にある適応室を訪れます。最初の主要なアップグレードにより、高圧の空気を呼吸できるようになり、500mを超える潜水に不可欠です。
  </Step>
  <Step num="4" title="バイオモッドのロックを解除">
    ハンマーヘッドやシーラッグなどのクリーチャーに**バイオスキャナー**を使用します。これにより、「暗闇で光る」や「泳ぐ速度の増加」などのパッシブアビリティがアンロックされます。
  </Step>
</Steps>

| 適応タイプ | 効果 | 要件 |
| :--- | :--- | :--- |
| **耐圧性** | ダメージなしで深く潜る | 初期難破船パビリオン |
| **消化器系** | エイリアン種を安全に食べる | ジェリープラントの根 |
| **耐熱性** | 溶岩バイオームで生き残る | 治療された世界樹の根 |
| **高度な視覚** | エイリアン（アクシオン）のテキストを読む | 第二感染樹 |
| **酸素ブースト** | O2消費量の減少 | バイオラボ研究 |

<Callout type="success" title="サバイバル必須アイテム">
常に**ソニックレゾネーター**を持ち歩きましょう。このツールは、より大きな鉱石の鉱脈を採掘するために必要であり、より深い物語の領域への進行を妨げる感染した樹の根を治療する唯一の方法です。
</Callout>

<Section icon={Shield} title="基地建設とオタマジャクシ型潜水艇">

**Subnautica 2オンライン**における基地建設は、協力的な取り組みです。新しいハビタットビルダーは、垂直通路や、車両用の**ムーンプール**のような専門的な部屋を含む、より複雑な構造を可能にします。初期から中期にかけての探索の主要な車両は、様々なシャシーモジュールでアップグレードできる多用途の潜水艇である**タッドポール**です。

<CardGrid cols={2}>
  <Card title="タッドポール潜水艇">
    - **容量**: 最大3人（パイロット1人、外部アームに2人）をサポート。
    - **カスタマイズ**: スカウトレイまたは有翼シャシーを装着可能。
    - **速度**: 高い機動性を持つが、頻繁なパワーセル交換が必要。
  </Card>
  <Card title="生息地のインフラ">
    - **電力**: 太陽光、バイオリアクター、水力タービン。
    - **ユーティリティ**: スキャナーステーション、改造ステーション、バッテリー充電器。
    - **ソーシャル**: 円卓の部屋と装飾ポスター。
  </Card>
</CardGrid>

電力管理は、マルチプレイヤー基地にとって最大の課題です。4人のプレイヤーがファブリケーターと充電器を使用している場合、単一のソーラーパネルでは不十分です。安定したエネルギー供給を維持するには、水中電流に設置された**水力タービン**または火山噴火口に設置された**熱発電機**に移行する必要があります。

| 車両モジュール | 機能 | 資源コスト |
| :--- | :--- | :--- |
| **深度モジュールMK1** | 深度制限の増加 | チタン、銀、ガラス |
| **スカウトシャシー** | 大幅な速度向上 | リチウム、硫黄 |
| **ストレージモジュール** | 外部貨物スペースの追加 | チタンインゴット |
| **有翼シャシー** | 操作性の向上 | 鉛、石英 |

<Callout type="info" title="電力のヒント">
**パワートランスミッター**を使用して、遠隔のエネルギー源（熱水噴出孔など）をメイン基地にリンクさせましょう。これにより、安全な浅瀬に基地を建設しながら、危険な高エネルギーバイオームから電力を引き出すことができます。
</Callout>

</Section>
