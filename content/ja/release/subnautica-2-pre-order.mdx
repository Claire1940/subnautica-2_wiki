## Subnautica 2の早期アクセスで期待できること

Subnautica 2は単なる続編ではなく、サバイバルフォーミュラの進化です。最初の2つのゲームは厳密にはシングルプレイヤー体験でしたが（サードパーティのMODを使用しない限り）、Subnautica 2はオプションの4人協力マルチプレイヤーをサポートするためにゼロから構築されました。ゲーム全体を一人でプレイすることもできますが、最大3人の友人と一緒にプレイする選択肢ができました。

### 新しい世界と新しい主人公
このゲームでは、「ノア」というパイオニアが登場します。彼の仕事は、皮肉な意味で「私の仕事はあなたの命です。都合の良い死に方を見つけてください」と説明されています。あなたはコロニー船が関わる災害の後、まったく新しい異世界の惑星に置き去りにされます。船のAIは、残骸にもかかわらず、あなたの任務を継続するよう主張します。この設定は、プレイヤーが植物、動物、古代の遺跡をスキャンすることで解決する必要がある謎を生み出します。

### 新しい乗り物とツール
「タッドポール」潜水艇がこのショーの新しい主役です。これは、以前のタイトルのシーモスに代わる、高度にカスタマイズ可能な初期の乗り物になるようです。早期アクセスが進むにつれて、開発者はより多くの乗り物、バイオーム、そして「バイオモッド」を追加する予定です。これらは、キャラクターを押しつぶされる深さに適応させる遺伝的または技術的なアップグレードです。

### 深海の危険
そびえ立つリヴァイアサンなしではSubnauticaではありません。トレーラーではすでに、プレイヤーを小さく見せる巨大な生物発光生物が予告されています。コミュニティの憶測では、これらの生物の一部は、個人に対してとは異なる反応をグループのプレイヤーに示す可能性があり、サバイバルゲームプレイに新たな戦術的深みが加わるとされています。

## PCシステム要件：実行できますか？

**Subnautica 2のプレオーダー**を確定する前に、あなたのPCが要件を満たしているかを確認することが不可欠です。幸いなことに、Unknown Worldsはゲームの最適化に定評があり、Subnautica 2の要件は最新のタイトルとしては驚くほど合理的です。

**最小要件:**
*   **OS:** Windows 10/11 64ビット
*   **プロセッサー:** Intel Core i5-4460 / AMD FX-8350
*   **メモリ:** 12 GB RAM
*   **グラフィックス:** Nvidia GeForce GTX 1050 Ti / AMD Radeon RX 470
*   **ストレージ:** 50 GB以上の空き容量

**推奨要件:**
*   **OS:** Windows 10/11 64ビット
*   **プロセッサー:** Intel Core i7-8700 / AMD Ryzen 5 2600
*   **メモリ:** 16 GB RAM
*   **グラフィックス:** Nvidia GeForce RTX 2060 / AMD Radeon RX 5700
*   **ストレージ:** 50 GB以上の空き容量 (SSDを強く推奨)

ゲームは12GBのRAMでも動作しますが、コミュニティの意見では、特に協力セッションをホストする場合、16GBの方がはるかにスムーズな体験を提供するとされています。より新しいエンジンへの移行により、照明と水物理学が以前の作品よりもGPUに負担をかけることになります。
