<Section icon={{Swords}} title="協力訓練の概要">

協力訓練は、ハンターの統計を大幅に向上させるために使用できる Solo Leveling: Arise の機能です。

<Callout type="info" title="ヒント">
協力訓練は、ゲームの序盤で最も重要です。より多くの訓練スロットを解放すると、CP を大幅に向上させ、より難しいコンテンツをクリアできるようになります。
</Callout>

### 訓練スロットの解放

訓練スロットは、以下の方法で解放されます。

- **ストーリーモードの進行：** ストーリーをクリアすると、新しい訓練スロットが解放されます。
- **ハンターレベルの上昇：** ハンターのレベルが上がると、追加の訓練スロットが解放されます。

### 訓練の種類

協力訓練には、さまざまな種類の訓練があり、それぞれ異なる統計ボーナスを提供します。

協力訓練でどの統計に焦点を当てるべきかを選択することは、CP を最大化し、効率的にコンテンツをクリアするために非常に重要です。

### 全般的な推奨事項

| 統計の種類 | 優先度 | 備考 |
| :------- | :----- | :--- |
| **攻撃力** | 高     | すべてのハンターにとって最も重要な統計です。常に攻撃力を優先してください。 |
| **致命打率** | 高     | 致命打を頻繁に発動することで、ダメージ出力を大幅に向上させます。 |
| **致命打ダメージ** | 高     | 致命打率と組み合わせることで、致命打の威力を最大化します。 |
| **貫通** | 中     | 特に防御力の高い敵に対して、ダメージ出力を向上させます。 |
| **集中** | 中     | スキルダメージを増加させます。スキルベースのハンターに最適です。 |
| **体力** | 低     | 序盤では有用ですが、終盤では他の統計ほど重要ではありません。 |
| **防御力** | 低     | 回避とシールドの方が、ダメージ軽減に効果的です。 |
| **回復** | 低     | ほとんどのハンターにとって、他の統計の方が優先されます。 |
| **抵抗** | 低     | 特定のコンテンツで役立つ場合がありますが、一般的には優先度が低いです。 |

協力訓練を最大限に活用するための追加のヒントと戦略をいくつか紹介します。

### 訓練のレベルアップ

- **均等なレベルアップ:** すべての訓練スロットを均等にレベルアップすることを目指してください。これにより、統計ボーナスのバランスが取れ、CP を着実に増加させることができます。
- **優先順位付け:** 最高の統計で説明したように、最も重要な統計を優先してください。
- **リソースの管理:** 訓練にはゴールドと訓練素材が必要です。これらのリソースを賢く管理して、効率的に訓練を進めてください。

### 覚醒

- **覚醒の重要性:** 訓練スロットを覚醒させると、統計ボーナスが大幅に増加します。
- **覚醒素材:** 覚醒には、特定の覚醒素材が必要です。これらの素材は、さまざまなゲームモードで入手できます。

### チーム構成

- **シナジー効果:** ハンターのチーム構成を考慮して、協力訓練の統計を調整してください。
- **役割のバランス:** ダメージディーラー、サポーター、タンクのバランスが取れたチーム構成を目指してください。
