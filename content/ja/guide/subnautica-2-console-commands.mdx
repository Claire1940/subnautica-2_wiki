### ステップバイステップのインストール

<Steps>
  <Step num="1" title="ゲームディレクトリの特定">
    Subnautica 2 のインストールフォルダに移動します。Steamユーザーの場合、ライブラリでゲームを右クリックし、「管理」から「ローカルファイルを閲覧」を選択します。
  </Step>
  <Step num="2" title="UE4SSのダウンロード">
    信頼できるModdingウェブサイトからSubnautica 2 用のUE4SSのzipファイルをダウンロードします。これは、他のModが機能するための基盤となるModです。
  </Step>
  <Step num="3" title="UE4SSファイルの解凍">
    ゲームディレクトリ内の `\Subnautica2\Subnautica2\Binaries\Win64` に移動します。ダウンロードしたUE4SSのzipファイルをここに置き、その内容をこのフォルダに直接解凍します。
  </Step>
  <Step num="4" title="コンソールコマンドModのダウンロード">
    コンソールコマンドModのzipファイルをダウンロードします。このModには実際のコンソール機能が含まれています。
  </Step>
  <Step num="5" title="コンソールコマンドModのインストール">
    コンソールコマンドModのzipファイルを開き、「DebugUIToggle」フォルダを見つけます。このフォルダ全体をゲームディレクトリ内の `\Subnautica2\Subnautica2\Binaries\Win64\Mods` に配置します。「Mods」フォルダが存在しない場合は作成します。
  </Step>
  <Step num="6" title="ゲームの起動とコンソールの開放">
    Subnautica 2 を起動します。ゲーム内で `F2` キーを押すと、コマンドコンソールが開きます。これで様々なオプションが表示され、コマンドを入力できるようになります。
  </Step>
</Steps>

### アイテムのスポーンと資源管理

コンソールコマンドの最も一般的な用途の1つは、任意のアイテムを直接インベントリにスポーンさせることです。これにより、資源の収集にかかる膨大な時間と労力を節約できます。

| コマンド構文              | 説明                                                            |
| :---------------------- | :-------------------------------------------------------------- |
| `item [名前] [数量]`    | 指定された数量のアイテムをインベントリに追加する。              |
| `spawn [名前] [数量]`   | 指定された数量のオブジェクトをプレイヤーの前にスポーンさせる。  |

**例:** インベントリに5個のリチウムを追加するには、`item lithium 5` と入力します。

基本的なチート以外にも、**Subnautica 2 のコンソールコマンド**は、デバッグ、環境制御、さらにはテレポートのためのツールを提供します。これらは、コンテンツクリエーター、モッダー、またはゲームのメカニズムを深く探求したいプレイヤーにとって特に役立ちます。

### テレポートコマンド

| コマンド            | 効果                                                              | 使用例                            |
| :------------------ | :------------------------------------------------------------------ | :------------------------------------ |
| `biome [名前]`      | プレイヤーを特定のバイオームにテレポートさせる。                  | `biome shallowtwistybridges`          |
| `warp [x] [y] [z]`  | プレイヤーを指定された座標にワープさせる。                          | `warp 100 50 200`                     |
| `goto [名前]`       | 名前付きの場所にテレポートする（例: `goto rocket`）。              | `goto rocket`                         |
| `spawnnearby`       | プレイヤーをランダムな方向に短い距離テレポートさせる（動けなくなった場合）。 | `spawnnearby`                         |

<Callout type="warning" title="座標系">
`warp` コマンドを使用する際は、`[x]` が東/西、`[y]` が高度、`[z]` が北/南であることを覚えておいてください。誤った座標は、危険な場所や意図しない場所に配置される可能性があります。
</Callout>
