## Configuration requise : pouvez-vous le faire tourner ?

Avant de vous lancer dans le **pré-chargement de Subnautica 2**, il est essentiel de vous assurer que votre PC peut supporter les environnements extraterrestres magnifiques mais exigeants. Unknown Worlds a fourni un détail des configurations minimales et recommandées. Le jeu présente des améliorations graphiques significatives par rapport à ses prédécesseurs, utilisant les dernières technologies de moteur pour restituer des éclairages sous-marins complexes et la bioluminescence.

### Configuration minimale (1080p, réglages bas)
*   **Système d'exploitation :** Windows 10/11 64 bits
*   **Processeur :** Intel Core i5-4460 / AMD Ryzen 3 1200
*   **Mémoire vive :** 8 Go de RAM
*   **Carte graphique :** NVIDIA GeForce GTX 960 / AMD Radeon RX 470
*   **Stockage :** 50 Go d'espace disponible

### Configuration recommandée (1440p, réglages élevés)
*   **Système d'exploitation :** Windows 11 64 bits
*   **Processeur :** Intel Core i7-9700K / AMD Ryzen 7 3700X
*   **Mémoire vive :** 16 Go de RAM
*   **Carte graphique :** NVIDIA GeForce RTX 3070 / AMD Radeon RX 6800
*   **Stockage :** SSD de 50 Go (fortement recommandé)

Pour ceux qui souhaitent pousser le jeu en **Ultra 4K**, les développeurs suggèrent au moins une RTX 4080 ou équivalent. De plus, il y a d'excellentes nouvelles pour les joueurs nomades : l'équipe a travaillé spécifiquement pour optimiser le jeu pour le **Steam Deck** et le **ROG Ally**. Bien que le statut officiel "Validé" puisse prendre quelques jours après le lancement, les premiers rapports suggèrent une expérience jouable sur ces appareils avec des réglages moyens.

## Nouvelles mécaniques de jeu à découvrir

Une fois que vous aurez terminé votre **pré-chargement de Subnautica 2** et que vous aurez plongé dans le jeu, vous remarquerez plusieurs changements majeurs dans le gameplay. L'une des fonctionnalités les plus discutées est le nouveau système de véhicules modulaires. Au lieu des véhicules statiques des jeux précédents, vous utiliserez désormais un système de "châssis".

Le "Tadpole" (Têtard), un petit engin de départ agile, peut être arrimé à divers modules de châssis qui modifient ses fonctionnalités. Cela permet un niveau de personnalisation jamais vu auparavant dans la série. Vous pouvez construire un châssis axé sur l'exploitation minière en eaux profondes, un autre pour la reconnaissance à grande vitesse, ou une version robuste pour le transport de fret.

La construction de base a également bénéficié d'une refonte "modulaire". Les joueurs peuvent désormais assembler des composants avec plus de liberté, créant des complexes sous-marins tentaculaires qui s'intègrent mieux au terrain. Que vous souhaitiez un observatoire confortable ou un immense centre industriel, les nouveaux outils sont conçus pour être plus intuitifs et créatifs.

## Foire aux questions (FAQ)

**Q : Puis-je jouer à Subnautica 2 hors ligne pendant l'Accès Anticipé ?**
R : Oui, une fois le jeu téléchargé et vérifié via Steam ou Epic Games, vous pouvez jouer au mode solo hors ligne. Cependant, vous aurez besoin d'une connexion internet pour les mises à jour et pour accéder à certaines fonctionnalités communautaires.

**Q : Existe-t-il une réduction si je possède les jeux précédents ?**
R : Bien qu'il n'y ait pas de réduction de fidélité permanente, les développeurs proposent souvent des packs "Complétez votre collection" sur Steam. Vérifiez la page de la boutique après le lancement pour voir si vous êtes éligible à un tarif préférentiel.

**Q : Comment forcer le pré-chargement de Subnautica 2 s'il ne démarre pas automatiquement ?**
R : Si vous avez pré-acheté le jeu sur Steam et que le téléchargement n'a pas commencé, essayez de redémarrer votre client Steam. Une fois redémarré, allez dans votre Bibliothèque, sélectionnez Subnautica 2, et le bouton "Installer" devrait maintenant être disponible.

**Q : Ma sauvegarde de l'Accès Anticipé sera-t-elle conservée pour la version 1.0 complète ?**
R : D'après l'expérience des titres précédents, Unknown Worlds fait de son mieux pour maintenir la compatibilité des sauvegardes. Cependant, en raison de la nature de l'Accès Anticipé et des modifications majeures du terrain, certaines sauvegardes peuvent devenir incompatibles ou présenter des bugs à l'approche de la sortie finale. Il est généralement recommandé de commencer une nouvelle partie pour profiter de la meilleure expérience possible lors du lancement de la 1.0.
